## Roles and responsibilities

Mobile engineer - Builds and maintains the mobile application, ensuring performance, stability, and platform compliance.  

QA (Quality Assurance) - Validates functionality, reliability, and regressions through manual and automated testing.  

Backend developer - Designs and implements server-side logic, APIs, and data persistence.  

Cybersecurity engineer - Protects systems and payment flows by designing security controls, monitoring threats, and ensuring compliance.  

Frontend developer - Implements user-facing web interfaces, focusing on usability, responsiveness, and integration with backend APIs.  

## Common skills across roles

Server-side programming, database design, API development, bug tracking, debugging, performance optimization.
